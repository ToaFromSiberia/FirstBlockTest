Здесь находится описание решения задачи.

take = []
give = []
Мы создали 2 пустых массива: массив "take" для сбора вводимых элементов и массив "give" для вывода элементов, удовлетворяющих условию задачи.

count = int(input('Введите кол-во элементов массива: '))
Здесь мы сообщаем программе сколько элементов мы собираемя ввести.

i = 0
while i < count:
    take.append(input('Введите элемент: '))
Вводим элементы. Они отправляются в массив "take".
    
    if len(take[i]) <= 3:
        give.append(take[i])
Введённый элемент проверяется, и если он соответствует условию, добавляется в массив "give".

    i += 1
Цикл продолжается, пока не будет введено указанное ранее количество элементов.

print(take)
Добавлять эту строку необязательно, но можно, если нужно посмотреть массив "take" целиком.

print(give)
Программа выводит конечный результат.
